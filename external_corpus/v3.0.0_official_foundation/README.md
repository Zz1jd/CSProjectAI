# v3.0.0 Official Foundation

This version is the governed control corpus for categories 1, 4, and 7.

Directory rule:
- Official foundation documents live under topic folders such as `theory/`, `instance_features/`, and `runtime_contracts/`.
- Journal section packages live under `journals/<journal_slug>/<year>_<first_author>_<paper_slug>/`.
- Template files end with `.md.example` and are excluded from manifest generation.

Current authored scope:
- `theory/` stores official CVRP modelling and destroy-repair notes.
- `instance_features/` stores benchmark-family and VRPLIB-format references.
- `runtime_contracts/` stores repository-sourced execution boundaries for the current FunSearch runtime.
- `journals/cor/2016_vidal_split_algorithm/` stores one complete governed journal package plus auxiliary flow and pseudocode nodes.

Starter template package:
- See `journals/_examples/ejor_2024_smith_adaptive_cvrp_study/` for a ready-to-fill sample skeleton.

Retrieval intent:
- This corpus is the stable control family for official theory, benchmark metadata, and hard runtime contracts.
